2. La importancia de la etiqueta es que el "h1" solo se puede utilizar una sola vez ya que se usa solo para mostrar el título. en cambio para usar los demas "h2, h3, etc" se pueden usar más de una vez ya que son encabezados.

11.Definir y ejemplificar las siguientes etiquetas: "blockquote", "cite", "hr", "code", "abbr" y "img"

11. <blockquote>: crea citas en bloque, marca las citas a otros autores o documentos.
ejemplo: <blockquote cite='http://html.conclase.net/w3c/html401...def-BLOCKQUOTE'>

Nota. Recomendamos que las implementaciones de hojas de estilo porporcionen un mecanismo para insertar signos de puntuación de citas antes y después de una cita delimitada por un BLOCKQUOTE de un modo apropiado según el contexto del idioma actual y el grado de anidamiento de las citas.

</blockquote>

<cite>: marca una referencia a una fuente, o el autor de un texto citado.
Ejemplo:

<cite>Galileo</cite> dijo: "... y sin embargo, se mueve."

<hr>: Representa un cambio de tema entre párrafos.

<code>:Es el apropiado para marcar el código de un programa.

<abbr title="">>: representa una abreviación o acrónico.

<img src="" alt="">: representa una imagen en el documento.
